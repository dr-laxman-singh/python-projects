# Simple Python Calculator

A basic command-line calculator built using Python. This project allows users to perform four fundamental arithmetic operations:

* Addition
* Subtraction
* Multiplication
* Division

## Features

* User-friendly menu-driven interface
* Supports decimal numbers
* Handles division by zero
* Validates operation selection
* Displays calculation results instantly

## Technologies Used

* Python 3

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/simple-python-calculator.git
```

2. Navigate to the project directory:

```bash
cd simple-python-calculator
```

3. Run the program:

```bash
calculator.py
```
